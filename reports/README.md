# Reports Directory

This directory stores lightweight reporting artifacts generated during the project lifecycle.

## Expected contents
- `figures/`: Publication-ready or documentation-ready visual outputs.
- `metrics/`: Lightweight JSON, CSV, or text summaries of model performance.
- final written summaries when appropriate.

## Versioning policy
Only commit concise and reproducible report artifacts. Avoid storing heavy intermediate outputs or duplicated result files.

## Current use
This directory already contains baseline evaluation metrics and will be extended as the project is consolidated.
